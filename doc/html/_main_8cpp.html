int main ( )

Entry point of the program.

Creates a Professor object and manages a list of WizardStudent objects. Through a menu-driven interface, the user can:

  • Create new wizard students
  • Teach spells to all students
  • Let students practice their spells
  • Exit the program

Dynamically created students are deleted before the program terminates to avoid memory leaks.

Returns
int Returns 0 when the program finishes successfully.
